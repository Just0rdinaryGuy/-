# Overview

Built a full-stack travel booking platform using modern web technologies. The goal was to create a seamless user experience for booking vacations.

## Key Features
- **User Authentication**: Secure login and signup.
- **Booking Management**: Real-time availability and booking history.
- **Admin Dashboard**: For managing destinations and orders.
